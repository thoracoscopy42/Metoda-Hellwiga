# Automatyzacja Metody Hellwiga 
Narzędzie do obliczania pojemności informacyjnej metodą Zdzisława Hellwiga na podstawie macierzy korelacji.

Celem programu jest ułatwienie studentom i badaczom wyboru optymalnych zmiennych objaśniających do modeli statystycznych.
# Jak używać programu
1. Przygotuj w Excelu macierz, która jest w formacie:
<img width="561" height="206" alt="image" src="https://github.com/user-attachments/assets/a027cd83-25fb-4010-9b44-bf4e47feebb6" />

Musi ona zawierać jedną kolumnę z nazwą Y - będzie to zmienna objaśniana.

2. Uruchom <b><i>Hellwig.exe.</i></b>

3. Wybierz plik excel i kliknij <i>Uruchom Metodę Hellwiga.</i>

4. W oknie aplikacji pojawi się 10 najlepszych rezultatów, a w nowym arkuszu zostaną dodane wszystkie obliczone kombinacje.

# Wymagania
Do użytkowania programu nie trzeba, poza samym plikiem <b><i>Hellwig.exe.</i></b>, nic pobierać.

Wszystkie wymagania do odtworzenia środowiska programistycznego znajdują się w pliku <b>requirements.txt</b>.
